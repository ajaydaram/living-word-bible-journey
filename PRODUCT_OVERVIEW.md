# The Living Word Bible Journey

## One-Page Product Overview

The Living Word Bible Journey is a guided, mobile-friendly Bible study app that helps people follow one clear path through Scripture instead of feeling lost in disconnected passages.

## Mission

Help people see the Bible as one unified redemptive story from Creation to Restoration, and build a sustainable daily reading habit.

## The Problem

Many readers want to study the Bible consistently but struggle with:
- Not knowing where to start.
- Losing continuity between stories.
- Feeling overwhelmed by too many study options.
- Dropping off after a few days.

## The Solution

This app provides a structured 100-story journey with a simple entry experience first, then deeper study tools when users are ready.

## 5W1H

### Who
- Individuals starting Bible study.
- Small groups and church communities.
- New believers needing a clear path.
- Returning readers who want consistency.

### What
- A web app for chronological Bible journey reading.
- Guided mode for simplicity and explore mode for deeper study.
- Story connections, thematic threads, and reflection prompts.

### When
- Daily devotion time.
- Weekly small-group prep.
- Sermon follow-up and discipleship practice.

### Where
- Any modern browser on phone, tablet, or desktop.
- Deployable as a public website with custom domain.

### Why
- To reduce confusion and increase consistency in Bible engagement.
- To move users from occasional reading to sustained spiritual habit.

### How
- React + TypeScript + Vite frontend.
- Curated 100-story local data model.
- localStorage-based progress persistence and resume flow.
- Scripture retrieval via Bible API with retry and external fallback.

## Key User Experience Highlights

- Quick start actions: Begin here, Resume, Deeper study.
- Last-read resume that survives page reload.
- Mobile dashboard collapse for lower cognitive load.
- Daily reflection prompt and streak motivation.
- Session summary to reinforce progress and next step.
- Scripture retry plus open-externally fallback when API fails.

## Why Churches and Groups Can Use It

- Makes first-time onboarding easier for members.
- Provides a shared story path across different maturity levels.
- Supports both simple devotional reading and advanced exploration.
- Works well for discipleship tracks and small-group continuity.

## Deployment Status

The app is production-ready for static hosting and already configured for Vercel deployment.

## Simple Pitch

The Living Word Bible Journey turns Bible study from overwhelming to actionable by giving every user one clear next story, one clear daily prompt, and one unified view of God’s redemptive narrative.
